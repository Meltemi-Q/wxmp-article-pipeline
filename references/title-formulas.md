# 标题公式

> 来源：writing-style.md §1，qclaw-diff-report.md 对比分析

---

## 核心原则

**更短，更有冲击力，有动词或形容词。**

---

## 好/坏对照表

| 原稿风格（避免） | 发布版风格（目标） |
|---|---|
| 白话叙事感 | 短句 + 亮点词 |
| "打开一看，这虾在摸鱼" | "工作室：这虾在摸鱼？" |
| "短信一响，内测通过了" | "内测短信：龙虾入场" |
| "微信打通了，这才是重点" | "微信直控：这才是必杀技" |
| "实测第一弹：搜个马斯克" | "实测 1：追着它搜马斯克" |
| "技能商店：需要什么装什么" | "技能商店：超能力中心" |
| "4000万 token 到底够不够用？" | "4000万 Token，真的够吗？" |

---

## 规律

- **动感词优先**："必杀技"、"超能力中心"、"翻烂了"
- **反差感**："够用吗？"、"真的假的？"、"这是坑吗？"
- **数字锚点**：实测 1、第 1 步、4000万
- **疑问句**：用问号结尾，引发好奇

---

## 官方限制

- 文章标题 ≤ **64 字**（超过 API 报 45003）
- 贴图标题 ≤ **20 字**

---

## 决策

| 场景 | 规则 |
|------|------|
| 标题太啰嗦 | 砍掉主语/连接词，只留核心信息 |
| 标题太平淡 | 加数字、反差词、问号 |
| 标题超过 64 字 | 拆成主副标题，或砍到 64 字内 |
