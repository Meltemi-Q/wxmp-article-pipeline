# 术语表

> 同一事物的不同叫法，上下文中可混用，不需要严格区分。

## AI 助手名称

| 名称 | 说明 |
|------|------|
| **小龙虾** | 主要用名，微信里叫"微信 ClawBot AI" |
| **ClawBot** | 微信里的实际显示名 |
| **OpenClaw** | 产品名，公众号/技术侧使用 |
| **微信小龙虾** | 微信场景下的叫法 |
| **AI** | 通俗叫法，写文章时通用 |

以上均指同一产品/助手，文章中可根据语境任选，读者能看懂即可。

## 技术术语（通俗解释）

| 术语 | 通俗解释 |
|------|---------|
| **Skill** | AI的"标准作业流程"，给它立个规矩让它知道遇到这类任务怎么做 |
| **反爬虫** | 网站的"防抓取机制"，防止机器自动访问 |
| **Cookie** | 登录凭证，相当于网站的"门禁卡"，证明你已经登录过 |
| **Whisper** | OpenAI的语音识别AI |
| **Gemini** | Google的多模态AI模型，能看图、能听音频 |
| **Playwright** | 浏览器自动化工具，可以模拟人操作浏览器 |
| **Jina Reader** | 把网页转成可阅读格式的工具 |
| **OCR** | 光学字符识别，把图片里的文字读出来 |
| **MCP** | Model Context Protocol，AI调用外部工具的协议 |

## 写作风格相关

| 术语 | 说明 |
|------|------|
| **PART** | 文章分段编号，后必须带描述（如"PART 1 Skill是什么"） |
| **Blockquote** | 引用块，用于金句、主观评价、核心结论 |
| **图注** | 每张图下方的说明文字，不能用文件名 |
