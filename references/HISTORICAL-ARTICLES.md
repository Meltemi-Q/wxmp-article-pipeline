## 读文章流程

写新文章前必须执行：

```bash
# 1. 查看最近发布文章列表
cat references/HISTORICAL-ARTICLES.md

# 2. 读最近3篇已发布文章（md格式）
cat references/archives/published/2026-04-09-我给AI装上了外挂它能自/content.md | head -80
cat references/archives/published/2026-04-06-顶级AI工具仍然离不开各/content.md | head -80

# 3. 确认写作风格一致后再动笔
```

## 拉取新文章

```bash
cd /root/.openclaw/skills/wxmp-article-pipeline
python3 scripts/archive_articles.py --latest        # 发布后立刻拉最新一篇
python3 scripts/archive_articles.py                  # 批量拉全部（跳过已存档）
python3 scripts/archive_articles.py --force          # 强制覆盖
```

## 对比原稿与发布版

```bash
cd /root/.openclaw/skills/wxmp-article-pipeline
python3 scripts/diff_articles.py --list               # 列出所有可对比的文章
python3 scripts/diff_articles.py "2026-04-09"       # 模糊匹配对比
python3 scripts/diff_articles.py "2026-04-09-完整文件夹名"
```
