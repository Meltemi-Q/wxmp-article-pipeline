# 禁用词 & 负面语气软化

> 来源：writing-style.md §6、§8

---

## 一、负面语气软化对照表

**删掉或软化具体批评性描述，保留行为观察。**

| 原稿写法（避免） | 发布版处理 |
|---|---|
| "速度确实有点慢，大概等了十几秒" | 删掉速度慢的描述 |
| "推脱，不肯动" | 改为"需要再催一句才动" |
| "资本家的血脉觉醒了哈哈哈" | 删掉"哈哈哈"，保留前半段改入 Blockquote |
| 提及马斯克"亲华言论"等政治敏感内容 | 删掉，只保留行为描述 |
| "内测嘛，这些都正常" | 删掉（自圆其说，不必说） |
| "速度有点慢" | 删掉或改中性描述 |

---

## 二、敏感内容处理

- **政治性描述**：只保留行为描述，删掉评价
- **涉及具体人的负面言论**：删掉或改行为描述
- **对比竞争对手**：只说事实，不说感受

---

## 三、语气太重时的通用软化方式

| 太重 | 软化 |
|------|------|
| "这是垃圾" | "这不是我的菜" / 直接删 |
| "完蛋了" | "这里有个坑" |
| "被坑了" | "这里要注意一下" |
| "傻子才用" | 删掉 |

---

## 决策

| 场景 | 规则 |
|------|------|
| 不知道怎么描述负面体验 | 只说行为（"被推脱了"），不说感受（"态度差"） |
| 不知道该不该提某个词 | 不肯定的 → 删，不确定事实的 → 删 |
| 有政治/争议性内容 | 删，只保留行为描述 |
